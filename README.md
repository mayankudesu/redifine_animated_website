## 📖 About the Project

**Redifine Animated Website** is a modern and visually stunning frontend project inspired by the creative experience of **Zentry.com**.

Built with **React.js**, **Tailwind CSS**, and **GSAP**, this project focuses on premium UI design, smooth animations, immersive transitions, and responsive layouts. It showcases modern frontend development skills with high-performance interactions and engaging user experiences.

---

## 🛠 Built With

### Tech Stack

- **React.js** – Frontend Library  
- **Tailwind CSS** – Styling Framework  
- **GSAP** – Animation Library  
- **Vite** – Build Tool  

---

### Key Features

- 🎨 Modern Awwwards-style UI  
- ⚡ Smooth GSAP animations  
- 📱 Fully responsive design  
- 🖱️ Interactive user experience  
- 🚀 Fast performance with Vite  
- ✨ Premium transitions & effects

## 💻 Getting Started

Follow the steps below to run the project locally.

---

### Prerequisites

- Node.js & npm  
- Git  
- Code Editor (VS Code recommended)

---

### Setup

Clone the repository:

```bash
git clone https://github.com/mayankudesu/redifine_animated_website.git
cd redifine_animated_website
```


### Install

Install dependencies:

```bash
npm install
```

### Usage

Start the development server:

```bash
npm run dev
```

Open your browser and go to `http://localhost:5173` (or the port shown in your terminal).

### Run Tests

> _No tests are currently set up. Add your test scripts here._

### Deployment

To build for production:

```bash
npm run build
```

To preview the production build:

```bash
npm run preview
```
